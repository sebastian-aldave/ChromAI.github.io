Ahora vamos a ver cómo hacer una lista ordenada de elementos. Para ello usamos las etiquetas

    y
  1. .

    1. Este es el primer elemento de la lista
    2. Este es el segundo elemento de la lista
    3. Este es el tercer elemento de la lista

    También podemos hacer una lista no ordenada, usando las etiquetas

      y
    • .

      • Este es un elemento de la lista no ordenada
      • Este es otro elemento de la lista no ordenada
      • Este es el último elemento de la lista no ordenada

      Para insertar una imagen en nuestra página, usamos la etiqueta , que tiene dos atributos: src, que indica la dirección de la imagen, y alt, que indica el texto alternativo que se muestra si la imagen no se carga.

      Por último, para crear un enlace a otra página web, usamos la etiqueta , que tiene el atributo href, que indica la dirección a la que queremos ir. El texto que va entre las etiquetas de apertura y cierre es el que se muestra como enlace.
